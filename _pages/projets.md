# Projets

## Analyse immobilière (DVF)

Analyse des transactions immobilières en France à partir de données ouvertes.  
Le projet vise à explorer les dynamiques de prix et à structurer les données pour en tirer des repères utiles.

Stack : Python, Pandas, API, Streamlit

Voir le dashboard : [dvf-analyse-immobiliere.streamlit.app](https://dvf-analyse-immobiliere.streamlit.app/)
